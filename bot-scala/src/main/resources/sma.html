Simple moving-average (SMA) Crossover Request 
Use following command to receive last month SMA for particular ticker:
/sma/{id}
Where:
id - ticker
Use following command to receive ticker SMA for concrete period:
/sma/{period}/{id}
Where:
period - period: 1m, 3m, 6m, 1y, 5y, 10y, 20y
id - ticker
As example:
/sma/3m/msft
