History Request 
Use following command to receive last month history for particular ticker:
/history/{id}
Where:
id - ticker
Use following command to receive ticker history for concrete period:
/history/{period}/{id}
Where:
period - period: 1m, 3m, 6m, 1y, 5y, 10y, 20y
id - ticker
As example:
/history/3m/msft
