Moving average convergence divergence (MACD) establishes the difference between a fast exponential moving average and a slow exponential moving average 
Use following command to receive last month MACD for particular ticker:
/macd/{id}
Where:
id - ticker
Use following command to receive ticker MACD for concrete period:
/macd/{period}/{id}
Where:
period - period: 1m, 3m, 6m, 1y, 5y, 10y, 20y
id - ticker
As example:
/macd/3m/msft
