Momentum (MOM) is an important measure of speed and magnitude of price moves. This is often a key indicator of trend/breakout-based trading algorithms 
Use following command to receive last month MOM for particular ticker:
/mom/{id}
Where:
id - ticker
Use following command to receive ticker MOM for concrete period:
/mom/{period}/{id}
Where:
period - period: 1m, 3m, 6m, 1y, 5y, 10y, 20y
id - ticker
As example:
/mom/3m/msft
