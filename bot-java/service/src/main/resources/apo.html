Absolute price oscillator (APO) is computed by finding the difference between a fast exponential moving average and a slow exponential moving average 
Use following command to receive last month APO for particular ticker:
/apo/{id}
Where:
id - ticker
Use following command to receive ticker APO for concrete period:
/apo/{period}/{id}
Where:
period - period: 1m, 3m, 6m, 1y, 5y, 10y, 20y
id - ticker
As example:
/apo/3m/msft
