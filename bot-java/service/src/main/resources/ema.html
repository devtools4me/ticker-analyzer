Exponential moving-average (EMA) Crossover Request 
Use following command to receive last month EMA for particular ticker:
/ema/{id}
Where:
id - ticker
Use following command to receive ticker EMA for concrete period:
/ema/{period}/{id}
Where:
period - period: 1m, 3m, 6m, 1y, 5y, 10y, 20y
id - ticker
As example:
/ema/3m/msft
