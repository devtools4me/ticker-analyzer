Buy Low Sell High (BLSH) Request 
Use following command to receive last month BLSH for particular ticker:
/blsh/{id}
Where:
id - ticker
Use following command to receive ticker BLSH for concrete period:
/blsh/{period}/{id}
Where:
period - period: 1m, 3m, 6m, 1y, 5y, 10y, 20y
id - ticker
As example:
/blsh/3m/msft
